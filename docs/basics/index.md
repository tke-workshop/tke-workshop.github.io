# 快速入门

本模块帮助您从零开始，快速掌握 TKE 的核心操作。

## 学习目标

完成本模块后，您将能够：

- [x] 准备 TKE 使用环境
- [x] 创建和管理 TKE 集群
- [x] 使用 kubectl 操作集群
- [x] 部署第一个应用

## 预计时间

| 章节 | 时间 |
|------|------|
| 环境准备 | 15 分钟 |
| 创建集群 | 20 分钟 |
| kubectl 基础 | 30 分钟 |
| 部署应用 | 25 分钟 |
| **总计** | **约 90 分钟** |

## 前置条件

- 腾讯云账号（[立即注册](https://cloud.tencent.com/register)）
- 开通 TKE 服务
- 本地安装 kubectl
