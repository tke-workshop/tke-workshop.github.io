# TKE Workshop 概览

TKE Workshop 是面向腾讯云容器服务 TKE 的云原生实践工作台。它把学习路径、操作指南、生产实践和可执行 Cookbook 放在同一个体验里，帮助工程师和 AI Agent 从“理解”走到“执行”和“验证”。

## 你可以怎样使用

| 目标 | 推荐路径 |
| --- | --- |
| 第一次接触 TKE | 先完成环境准备，再进入 Operate 路径 |
| 需要完成具体操作 | 按集群、节点、工作负载、Service 查找任务 |
| 建设生产集群 | 从 Practice 路径选择安全、可用性、扩展、成本或升级主题 |
| 验证自动化脚本 | 进入 Cookbooks，选择可执行 Recipe |
| 构建 AI 工作负载 | 进入 AI on TKE 路径 |

## Agent-First 原则

每个操作型内容都应该包含：

- 清晰的任务目标。
- 前置条件和输入参数。
- 可复制的命令、SDK 或 YAML 示例。
- 验证步骤和期望结果。
- 常见错误与处理建议。
- 可交给 Agent 的 Prompt 模板。
