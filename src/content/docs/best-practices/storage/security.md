# 存储安全

存储安全覆盖数据加密、访问权限、Secret 管理、误删保护和审计。

## 基线建议

- 生产 PVC 使用 `Retain`，删除前人工确认。
- COS 使用子账号和最小权限，不使用主账号密钥。
- Secret 不写入 Git、镜像和日志。
- 对 CBS、CFS、COS 启用合适的加密和访问控制。
- 对关键数据配置快照、版本控制或备份。

## 权限边界

| 层面 | 控制方式 |
|------|----------|
| Kubernetes | RBAC 限制 PV/PVC/Secret 操作 |
| 云资源 | CAM 策略限制 CBS/CFS/COS 操作 |
| 网络 | CFS 权限组、VPC、安全组、ACL |
| 数据 | 加密、备份、版本控制 |
