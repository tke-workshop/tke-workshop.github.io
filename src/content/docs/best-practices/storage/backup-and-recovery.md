# 备份与恢复

持久化不等于备份。PVC 只保证卷生命周期，不保证误删、误写、勒索或应用逻辑错误后的恢复。

## 建议策略

| 数据类型 | 备份方式 |
|----------|----------|
| CBS 数据盘 | 云硬盘快照、应用一致性备份 |
| CFS 文件 | 文件系统快照、定期同步到 COS |
| COS 数据 | 版本控制、生命周期、跨地域复制 |
| Kubernetes 资源 | 集群备份工具、GitOps 声明文件 |

## 恢复演练

- 定期从快照创建新卷并启动验证 Pod。
- 验证数据库恢复到可读写状态，而不是只验证卷能挂载。
- 记录 RPO/RTO，明确谁能执行恢复。
- 备份任务避开业务高峰，避免影响 I/O。
