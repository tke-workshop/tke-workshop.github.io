# 生产实践路径

Practice 路径面向生产集群治理。它不是单个功能教程，而是一组可以组合执行的工程实践。

## 安全

- 集群加固。
- 身份认证和访问管理。
- Pod 安全和运行时隔离。
- 镜像安全、数据安全、合规与审计。

## 可用性

- etcd 过载保护。
- apiserver 限速和熔断。
- 防误删策略。
- 备份与恢复。

## 可扩展性

- 控制平面扩展。
- 数据平面扩展。
- 系统组件扩展。
- 工作负载扩缩容。

## 成本优化

- 成本分析。
- 调度优化。
- 算力碎片规整。
- 动态伸缩。

## 升级

- 控制面升级。
- 数据面升级。
- 组件升级。
- 跨版本迁移。
